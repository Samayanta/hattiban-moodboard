## 🚀 Deploy Options

### GitHub Pages (Recommended - Free)
```bash
# 1. Create repo on GitHub
git init
git add .
git commit -m "Initial moodboard"
git remote add origin https://github.com/YOUR_USERNAME/hattiban-moodboard.git
git push -u origin main

# 2. Go to Settings → Pages → Deploy from main branch
```

### Netlify (Easiest)
1. Go to [netlify.com/drop](https://netlify.com/drop)
2. Drag & drop the `/moodboard` folder
3. Done - instant URL

### Vercel
```bash
npm i -g vercel
vercel --prod
```
